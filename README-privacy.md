# Eat&Run Privacy Policy

This repository hosts the privacy policy for the Eat&Run mobile application.

## GitHub Pages Setup

This privacy policy is hosted on GitHub Pages. To access it:

1. Go to your repository settings
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select the branch (usually `main` or `master`)
4. Click "Save"
5. Your privacy policy will be available at: `https://manalorojohn-ai.github.io/eatrun-privacy/privacy-policy.html`

## File Structure

- `privacy-policy.html` - The main privacy policy document

## Usage

Use this URL in Google Play Console:
